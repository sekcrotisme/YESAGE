YES AGE

YOCTO Arcade Games adalah game edukasi untuk semua kalangan dengan visi meningkatkan ketangkasan dalam berpikir dan menyusun strategi. Game ini dirancang untuk menambah kreativitas dan memberi reward berupa poin AGE yang dapat dikonversi menjadi koin YOCTO di level selanjutnya.

Tingkatan Level:
1. DrakAGE
2. LightAGE
3. GoldenAGE
4. GloryAGE
5. YesAGE (Final Evolution)

Basic game: Arcade + Puzzle + Task-based + Blockchain Quiz
Edukatif, Menantang, dan Menyenangkan.
